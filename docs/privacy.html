碎片 隐私政策

最后更新：2026 年 5 月 28 日

碎片（Suipian）是一款个人日记与碎片记录工具。 我们非常重视你的隐私——我们不收集、不出售、不共享你的任何个人数据。 你所记录的一切内容仅保存在你的设备本地，并可选择通过 Apple iCloud 在你自己的账号下同步。

我们访问哪些权限

权限用途数据去向
相机 拍摄照片并附加到碎片 仅保存在你的相册和设备本地
麦克风 录制语音备忘 仅保存在设备本地
照片库 从相册选取图片或视频附加到碎片 仅保存在设备本地
位置 记录碎片的拍摄地点（可选） 仅保存在设备本地，不上传至任何服务器
语音识别 将语音转换为文字快速输入 由 Apple 系统 API 处理，受 Apple 隐私政策约束
Face ID / Touch ID 解锁私密碎片，保护隐私内容 仅在设备本地验证，我们无法访问生物识别数据
Apple Music 读取当前正在播放的歌曲信息（可选附加） 仅保存在设备本地

iCloud 同步

碎片支持通过 Apple iCloud（CloudKit）在你的多台设备之间同步数据。 同步功能完全在你的 Apple 账号私有数据库中进行， 我们的开发者无法访问你的 iCloud 内容。 你可以随时在「设置 → Apple ID → iCloud → 管理账户存储」中关闭或删除碎片的 iCloud 数据。

我们不做的事

  • 不注册账号，不需要填写任何个人信息
  • 不收集使用行为数据或统计分析数据
  • 不接入任何第三方广告 SDK
  • 不将你的任何数据上传到我们的服务器（我们没有服务器）
  • 不出售或共享你的数据给任何第三方

第三方服务

碎片仅使用 Apple 官方框架（SwiftData、CloudKit、Speech Framework、PhotosUI 等）， 不集成任何其他第三方 SDK 或服务。 Apple 相关服务的隐私政策请参阅 Apple 隐私政策。

儿童隐私

碎片不专门面向 13 岁以下儿童，也不主动收集任何用户数据，因此对所有年龄层用户同等适用上述政策。

政策变更

如果本政策发生重大变化，我们会在应用更新说明中注明，并更新本页面顶部的日期。 继续使用碎片即表示你接受更新后的政策。
