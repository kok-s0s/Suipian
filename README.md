# 碎片

一款用于记录生活片段的 iOS 日记应用，基于 SwiftUI + SwiftData 构建。

## 功能

### 四大主视图

| 标签页 | 功能 |
|--------|------|
| 碎片 | 主信息流，支持列表 / 瀑布流网格切换 |
| 故事线 | 将碎片归入不同故事线，按时间串联 |
| 地图 | 带有聚合算法的位置地图，可跳转到具体碎片 |
| 统计 | 记录热力图、连续天数、情绪趋势、标签分布等 |

### 碎片记录

每条碎片支持记录：

- 文字内容
- 图片 / 视频（来自相册或直接拍摄）
- 情绪标签（emoji）
- 位置（GPS 坐标 + 地名）
- 自定义标签
- 故事线归属
- 音频
- 正在播放的音乐（元数据）
- 私密 / 置顶标记

### 信息流交互

- 按标签筛选，支持快速切换
- 全文搜索（内容、标签、地点、故事线、情绪）
- 时间升 / 降序排序
- 置顶碎片浮于顶部
- **随机回顾**：随机翻出一条 7 天以前的碎片
- **今日历史**：展示往年同一天记录的碎片

### 统计

- 总数 / 本月 / 近 7 天汇总卡片，点击可钻取查看对应碎片
- 连续记录天数（Streak）
- 13 周记录热力图，点击格子查看当日碎片
- 近 14 天情绪时间轴
- 情绪分布气泡图
- 常用标签流式排列，TOP 12

### 其他

- **App 锁定**：切后台自动上锁，支持 Face ID / Touch ID 解锁
- **每日提醒**：可设置自定义时间的每日推送通知
- **数据导出**：将所有碎片元数据导出为 JSON 文件（不含媒体）
- **桌面小组件**：在主屏幕展示最近一条碎片（小、中号）
- **Spotlight 搜索**：碎片内容可在系统全局搜索中找到
- **分享卡片**：将碎片生成为可分享的图片卡片
- **背景纹理**：可选点阵、斜纹、方格纹理

## 技术栈

- **SwiftUI** — 声明式 UI，iOS 17+
- **SwiftData** — 本地持久化
- **MapKit** — 地图与位置聚合
- **PhotosUI / AVFoundation** — 照片选择、相机、音频
- **LocalAuthentication** — 生物识别锁定
- **UserNotifications** — 本地每日提醒
- **WidgetKit** — 主屏幕小组件
- **Core Spotlight** — 系统级搜索集成

## 构建

需要 Xcode 15+，iOS 17+ 真机或模拟器。

```
open Suipian.xcodeproj
```

小组件和 Spotlight 功能需要 App Group 与对应 entitlement，Bundle ID 为 `com.kok-s0s.Suipian`。
